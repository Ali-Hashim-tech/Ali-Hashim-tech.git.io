Chapter 1: Getting Started with Python

1.1 Installing Python

To start programming in Python, you need to install Python on your computer. Here’s how you can do it:

  1. Download Python: Go to the official Python website and download the latest version of Python suitable for your operating system.
  2. Run the Installer: Open the downloaded file and follow the installation instructions. Make sure to check the box that says “Add Python to PATH” before clicking “Install Now.”
  3. Verify Installation: Open your terminal or command prompt and type python --version to verify that Python is installed correctly.

1.2 Writing Your First Python Program

Let’s write a simple Python program to get started:

  1. Open a Text Editor: Use any text editor or an Integrated Development Environment (IDE) like PyCharm or VS Code.
  2. Write the Code: 
    print("Hello, World!")
  3. Save the File: Save the file with a .py extension, for example, hello.py.
  4. Run the Program: Open your terminal or command prompt, navigate to the directory where your file is saved, and type python hello.py to run your program.

Chapter 2: Basic Python Concepts

2.1 Variables and Data Types

Python supports various data types, including integers, floating-point numbers, strings, and more. Here’s how to declare variables:

# Integer
x = 5

# Float
y = 3.14

# String
name = "Alice"

2.2 Basic Operators

Python includes several operators for performing arithmetic operations:

# Addition
sum = 5 + 3

# Subtraction
difference = 5 - 3

# Multiplication
product = 5 * 3

# Division
quotient = 5 / 3

2.3 Conditional Statements

Use if, elif, and else to perform conditional operations:

x = 10

if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is 5")
else:
    print("x is less than 5")

2.4 Loops

Python supports for and while loops:

# For loop
for i in range(5):
    print(i)

# While loop
i = 0
while i < 5:
    print(i)
    i += 1

Chapter 3: Functions in Python

3.1 Defining Functions

Functions in Python are defined using the def keyword. Here’s a basic example:

def greet(name):
    return "Hello, " + name

print(greet("Alice"))

3.2 Function Parameters and Arguments

Functions can take parameters and arguments. Here’s how to define a function with parameters:

def add(a, b):
    return a + b

result = add(5, 3)
print(result)

3.3 Return Values

Functions can return values using the return statement. Here’s an example:

def square(number):
    return number * number

print(square(4))

3.4 Scope of Variables

Variables defined inside a function are local to that function. Here’s an example:

def local_variable_example():
    x = 10  # Local variable
    print(x)

local_variable_example()
print(x)  # This will cause an error because x is not defined outside the function

3.5 Lambda Functions

Lambda functions are small anonymous functions defined with the lambda keyword:

square = lambda x: x * x
print(square(5))
